JavaScript reduce用法

reduce 是 JavaScript 数组的一个高阶函数方法，用于对数组中的每个元素执行一个reducer函数，将其结果汇总为单个返回值。

基本语法

javascript array.reduce(callback(accumulator, currentValue, currentIndex, array), initialValue)

参数说明

  1. callback - 执行每个数组元素的函数，包含四个参数：
    • accumulator (累计器)：上一次调用回调函数返回的值，或initialValue - currentValue (当前值)：当前正在处理的数组元素
    • currentIndex (当前索引)：当前正在处理的数组元素的索引
    • array (源数组)：调用reduce的数组本身
  2. initialValue (可选)：作为第一次调用callback函数时的第一个参数的值

工作原理

  • 如果提供了initialValue，则accumulator初始值为initialValue，从数组第一个元素开始迭代
  • 如果没有提供initialValue，则accumulator初始值为数组第一个元素，从第二个元素开始迭代
