605. 种花问题 - 力扣（LeetCode）

题目描述

输入一个由0和1组成的数组，0表示没有种花，1表示种花了，两个1不能相邻，问：能否在当前情况下种下n朵花？

输入输出样例

题目解释

要保证空间利用率最大。f(n)表示下标为n时，能不能种花，

前提条件都是flowerbed[n] == 0

$$
f(n)=\begin{cases}
1, flowerbed[n-1] == 0 && flowerbed[i + 1] == 0\\
0, flowerbed[n-1] != 0 || flowerbed[i+1]!=0
\end{cases}
$$
