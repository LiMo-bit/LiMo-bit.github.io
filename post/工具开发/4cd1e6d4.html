yaml编辑器开发

发表于2025-11-01|更新于2025-11-09|工具开发
|浏览量:

设计说明

  • 目标
    建立一个可视化编辑 hexo yaml 文件的工具

  • 功能—以 todolist.yaml 为例

    以类似 excel 的方式管理 todolist

    • 支持新建计划表，支持创建子计划表
    • 支持修改当前计划状态
    • 支持展示当前计划情况

  • 开发步骤

    • 处理 yaml 文档，生成更方便 Python 的格式内容
      • Python 读取 yaml 文件
      • 【进阶】Python 提炼 yaml 文件元素，并输出模版
      • Python 生成 yaml 中 task 的管理类
    • tkinter 处理
      • 生成有数据 UI 界面
        • 生成空白 UI 界面
        • 【进阶】根据模板生成 UI 界面
        • 像 UI 界面填充数据
      • 支持操控 UI 界面
        • 支持右键添加/删除任务
        • 支持复选框完成任务
        • 支持双击修改任务内容
        • 支持移动任务顺序—支持设置任务优先级
        • 支持约束条件设置
        • 支持双击展开/收回任务

【约束】
  • 支持一键完成任务，完成父任务的同时，完成子任务(位操作?)
  • 子任务不完成，父任务也不能完成
  • 任务完成时间不能大于任务创建时间
  • 子任务创建时间不能大于父任务创建时间 → 重构->双向指针
  • 任务不完成，任务完成时间应该清空
