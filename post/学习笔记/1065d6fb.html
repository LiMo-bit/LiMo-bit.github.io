CSS 介绍

规则

  • 除了选择器部分，每个规则集都应该包含在成对的大括号里（{}）。
  • 在每个声明里要用冒号（:）将属性与属性值分隔开。
  • 在每个规则集里要用分号（;）将各个声明分隔开。

selector

选择器类型

CSS 选择器 - 学习 Web 开发 | MDN

选择器名称 选择内容 实例
元素选择器(标签或者类型选择器) 指定类型的 HTML 元素 p
选择 < p >
ID 选择器 具有特定 ID 的元素，单一 HTML 页面中，ID 和元素一一对应 #my-id
选择<p id='my-id'>或者<a id='my-id'
类选择器 具有特定类的元素。单一页面中，一个类可以有多个实例，一个元素也可以有多个类。
类的命令不能是中文		 .my-class
选择<p class="my-class">或者<a class="my-class">
属性选择器 具有特定属性的元素 img[src]
选择<img src="aaa.jpg">但是不会选中<img>
伪类选择器 特定状态下的特定元素(比如鼠标指针悬停在链接上面时) a:hover
选择尽在鼠标指针悬停在链接上时的<a>元素
伪元素选择器 选择一个元素的某个部分 p::first-line
选择的是p标签的第一行
关系选择器 选择某种特定继承关系的标签 article > p
选中article的直接子代元素

属性选择器

  • 存否和值选择器

    允许基于一个元素自身是否存在或者基于各式不同的按属性值的匹配来获取元素

    选择器 实例 描述
    attr a[title] 匹配带有一个名为attr的属性的元素——方括号里的值
    [attr=value] a[href="https://example.com"] 匹配带有一个名为attr的属性的元素，其值正为value——引号中的字符串。
    [attr~=value] p[class~="special"] 匹配带有一个名为attr的属性的元素，其值正为value，或者匹配带有一个attr属性的元素，其值有一个或者更多，至少有一个和value匹配。
    注意，在一列中的好几个值，是用空格隔开的。
    [attr|=value] div[lang|=”zh”] 匹配带有一个名为attr的属性的元素，其值可只为value*，或者开始为*value，后面紧随着一个连字符。
    如果是attr为class，则匹配类属性仅为value或者类属性中有以value开头加连字符的

  • 子字符串匹配选择器

    选择器 示例 描述
    [attr^=value] li[class~="box-"] 匹配带有一个名为attr的属性的元素，其值开头为value子字符串
    [attr$=value] li[class$="-box"] 匹配带有一个名为attr的属性的元素，其值结尾为value子字符串
    [attr*=value] li[class*="box"] 匹配带有一个名为attr属性的元素，其值的字符串中的任何地方，至少出现了一次value子字符串

    这个觉得挺简单的，不需要例子了

  • 大小写敏感
    如果需要以大小写不敏感的方式匹配属性值，需要在闭括号之前，使用i值，例如li[class^=a i]这样匹配以a开头的class属性时，会考虑大写A，

关系选择器

关系选择器通过和周边标签的关系定位到本标签
关系选择器 - 学习 Web 开发 | MDN

选择器 实例 描述
后代选择器 body article p body下的article下的p标签
只要后代有，不管是第几代
子代关系选择器 article > p 作为article直接子类的p标签
必须是亲儿子
邻接兄弟选择器 p + img p和img同级，且img紧跟p之后
img是老二，p是老大
通用兄弟选择器 p ~ img p和img同级，且img比p小
img是老二三四，p是老大
